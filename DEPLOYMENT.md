# Laravel Jobs Portal Deployment on AWS EC2

## Project Overview
This project is a Laravel jobs portal deployed on an AWS EC2 Ubuntu server using Nginx and PHP-FPM.

## Stack Used
- AWS EC2
- Ubuntu Linux
- Nginx
- PHP 8.3-FPM
- MySQL
- Laravel
- Git/GitHub
